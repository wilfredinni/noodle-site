We are excited to launch the official blog for **Noodle**! 

Noodle is a terminal-based REST client that lets you inspect, send, and iterate on HTTP requests using simple YAML files on disk.

## Why Noodle?

Working with API clients often forces you to choose between graphical interfaces that clutter your workspace, or complicated `curl` commands that are hard to version control and iterate on. Noodle bridges this gap by letting you define requests in plain text YAML, and interact with them using a slick, keyboard-first terminal interface.

Stay tuned for more updates, tutorials, and feature releases!
