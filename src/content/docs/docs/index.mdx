Welcome to the Noodle documentation. Noodle is a terminal REST client
that treats HTTP requests as files on disk: inspect, send, and iterate
from the TUI.

## Getting Started

- [Installation](/docs/getting-started/installation/): install via curl, Homebrew, or source
- [Quick Start](/docs/getting-started/quick-start/): send your first request in under a minute
- [Concepts](/docs/getting-started/concepts/): collections, requests, environments, and more
- [CLI Commands](/docs/getting-started/cli/): TUI, import, update, and automation commands
