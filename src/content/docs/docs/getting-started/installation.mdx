Get noodle on your machine in under a minute. Three ways, take your pick.

## Quick install

```bash
curl -LsSf https://noodlerest.dev/install.sh | sh
```

Detects your OS and architecture, downloads the latest binary from GitHub
Releases, and drops it at `~/.local/bin/noodle`.

If `~/.local/bin` isn't on your `$PATH`, the script will let you know: just
add this to your shell config (`~/.zshrc`, `~/.bashrc`, etc.):

```bash
export PATH="$HOME/.local/bin:$PATH"
```

**Custom install location?** Set `NOODLE_INSTALL_DIR`:

```bash
NOODLE_INSTALL_DIR=/usr/local/bin curl -LsSf ... | sh
```

**Need a specific version?** Pin it with `NOODLE_VERSION`:

```bash
NOODLE_VERSION=v0.1.0 curl -LsSf ... | sh
```

## Homebrew

```bash
brew tap wilfredinni/noodle
brew trust wilfredinni/noodle
brew install noodle
```

## From source

Requires [Bun](https://bun.sh).

```bash
git clone https://github.com/wilfredinni/noodle
cd noodle
bun install
bun run build:bin
```

The binary lands at `./noodle`. Move it somewhere on your `$PATH`.

## Did it work?

```bash
noodle --help
```

You should see the CLI usage output. To launch the TUI:

```bash
noodle
```

## Supported platforms

Prebuilt binaries for these combos: others need to build from source.

| Platform | Architecture | Binary |
|----------|-------------|--------|
| macOS    | arm64       | ✅     |
| Linux    | x86_64      | ✅     |
| Linux    | arm64       | ✅     |

Running an Intel Mac? Build from source above.

## Updating

```bash
noodle update
```

Homebrew installs redirect to `brew upgrade noodle`. Everything else
downloads and replaces the binary in-place.
