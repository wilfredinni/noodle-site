---
title: Responses
description: Reading and working with responses
---

After sending a request, the response pane displays:

- **Status code** and status text
- **Response time** in milliseconds
- **Headers** (sorted alphabetically)
- **Body** (formatted JSON or plain text)
- **Body size**

## Copy Response Body

Press **Ctrl+B** to copy the response body to your clipboard.

## Timeline

Each response includes timing information for the full request lifecycle.

## Error Handling

Noodle displays errors inline in the response pane when:
- The server is unreachable
- DNS resolution fails
- A timeout occurs
- The response is malformed
