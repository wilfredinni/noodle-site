This release brings a command palette for quick actions, runtime collection switching, and a generalized picker component powering multiple overlays.

## ✨ New

- 🎛️ **Command Palette** (`Ctrl+P`): section-grouped command palette with fuzzy filtering and direction-aware navigation
- 📂 **Collection Switcher** (`Ctrl+O` or command palette): switch between previously-used collection directories at runtime, persisted with MRU ordering
- 🧩 **PickerOverlay**: reusable picker overlay component now drives both the command palette and theme picker

## 🎮 Keybinding Changes

- `env_cycle` moved to `Ctrl+U` (was `Ctrl+Alt+P`) to avoid terminal modifier ambiguity
- `command_palette` now on `Ctrl+P`

## 🐞 Fixes

- Clipboard: OSC52 fallback for terminals without binary clipboard support
- Clipboard: handles large response bodies without truncation on copy
- Clipboard: synchronous write prevents race conditions with env editor focus
- Env editor: default focus lands on env-header instead of sidebar
- Command palette: up-arrow wrap from first item works on first keypress
- Theme picker (`Ctrl+T`): now works from folder focus in the sidebar
- PickerOverlay: stale closure in keyboard nav fixed, interceptor no longer blocks input when hidden, search resets on close
- Collection switcher: MRU fallback on startup, stale directory reference in switch callback, double-parse race in cmd-palette bypass

## 🔧 Refactors

- Command palette commands extracted to `src/ui/commands.ts`
- Status bar now shows environment and palette shortcut hints
