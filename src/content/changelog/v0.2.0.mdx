This release brings a global undo command, consistent variable highlighting across all inputs, and a more reliable theme switcher.

## ✨ New

- ↩️ **Undo all with Ctrl+Z**: revert every unsaved draft change across requests, folders, and environments in one keystroke
- 🎨 **Unified `$VAR` coloring**: every input field now uses the same syntax highlighting for environment variables
- 💾 **Persistent themes**: switching themes no longer breaks when the built-in theme list changes

## 🐞 Fixes

- Edit overlay no longer opens when a folder is selected in the sidebar
- Loading spinner centered in the response pane
- Stale UI state cleared when reverting drafts
- URL field focuses correctly in the New Request overlay
