This release fixes overlay key event leaks, adds variable substitution to timeline entries, improves sidebar cursor behavior, and refactors folder auth handling.

## 🐞 Fixes

- **Overlay key isolation**: key events from help, env editor, and theme picker overlays no longer leak through to response pane scroll or pane focus cycling
- **Timeline variable substitution**: `$VAR` placeholders in timeline entries now resolve against the active environment for body, headers, params, and auth
- **Sidebar cursor stability**: saving a folder no longer jumps the cursor to a request row; `Ctrl+E` editing a request no longer resets sidebar focus to the first entry
- **Save-toast readability**: toast now shows the request name instead of the file path ID
- **Folder activity accuracy**: OK% counts only 2xx/3xx responses as success instead of any response with a status code
- **Auth type switching**: switching auth type in a folder no longer loses existing auth values
- **Import tips**: command-line tips reference `noodle import` instead of `bun run import`

## 🔧 Refactors

- Removed `inherit` from folder auth type — folders now always resolve auth directly
